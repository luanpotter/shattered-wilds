Welcome, Hero, to Shattered Wilds.

Shattered Wilds is a work-in-progress base setting for a new TTRPG system based on D12 dice rolls.

Heavily inspired by D&D 5e, accidentally similar to Pathfinder 2e, with some elements of Fate, D&D 3.5, and some original ideas.
